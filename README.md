## Development

### Requirements

Node.js >= 20.0.0
  
### Configuration

```sh

npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```
served on http://localhost:5173

## Production & deployment

Continuous integration with Firebase Hosting by Google.
Use of Firebase Auth to handle user authentification.
